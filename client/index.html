Self Driving Car & Image Recognition

Our goal was to create a software with Arduino that provides a way for the user to parallel park the car, and move the car around via another device if it’s in manual mode. The user should be able to swap to automatic mode at any point, which will make it look for a parking space. The user should be able to make the car move forward and include obstacle detection such as changing direction if obstructed by an obstacle. If the automatic button is pressed the car should find a suitable parking space to the right which is indicated by detecting two obstacles with a space between them and parking between them while also not running into said obstacles.

Our product:

  • Autonomous parallel parking of Arduino car
  • Manual driving of Arduino car using our app
  • Color detection using raspberry pi
  • Adjust the speed of Arduino car using our app
  • Streaming + tilting phone to drive the Arduino car using our app
  • Native android application

Gituml

Our system is the extension of Github. A website where users can login with their git account and choose which repository they want to convert to the UML diagram. users will be able also to change class names or add notes to the diagram. the system will be blackboard style because all the contributors in the same repository in the Github will be able to see the same diagram and see the changes and notes made by other users which whom he shares the repository with.

Our product:

  • Distributed system
  • blackboard architecture style with middleware
  • More computers added to our network more processing and database power we get
  • If one of the servers has crashes the system will automatically configure the connections
  • User will use our system to visualize the source code into class and force diagram
  • User can add a comment and make changes to the diagram
  • Users that are contributors on the same repository will see all the changes made by other users
  • context of the page will automatically be updated without needing to refresh the page "Long pulling"
